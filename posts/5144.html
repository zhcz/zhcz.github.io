书法字提取专业版 (Calligraphy Extractor Pro) - 使用说明书

感谢您购买/试用本软件！本工具专为书法爱好者和专业人士设计，能从复杂的图片中批量提取、去噪并整理书法字体。

1. 安装与运行

本软件为绿色免安装版。

Windows 用户

  1. 解压您收到的压缩包。
  2. 进入文件夹，找到 CalligraphyExtractor.exe。
  3. 双击即可运行。

macOS 用户

  1. 解压您收到的压缩包。
  2. 找到 CalligraphyExtractor.app。
  3. 重要提示：首次运行时，如果提示“无法打开”或“未识别的开发者”：
    • 请右键点击图标，选择 **“打开” (Open)**。
    • 在弹出的对话框中再次点击 “打开”。
    • （这是因为软件未上架 App Store，只需操作一次，以后双击即可）。

2. 试用版 vs 专业版

软件默认以试用模式运行，您可以免费体验核心功能。

功能 试用版 (Trial) 专业版 (Pro)
基础提取 ✅ 支持 ✅ 支持
实时预览 ✅ 支持 ✅ 支持
批量处理 ⚠️ 每次仅限前 5 张 ✅ 无限制
高级参数 ❌ 锁定不可用 ✅ 完全开放 (膨胀、合并距离等)
去噪微调 ❌ 默认设置 ✅ 可自定义

3. 如何注册激活

如果您已购买专业版，请按照以下步骤激活：

  1. 打开软件，点击菜单栏的 Settings (设置) -> **Register Pro Version (注册专业版)**。
  2. 在弹出的窗口中输入：
    • User Info (用户信息)：输入您购买时提供的邮箱或手机号。
    • **License Key (激活密钥)**：输入卖家发给您的长字符串。
  3. 点击 OK。
  4. 提示注册成功后，重启软件即可解锁所有功能。

提示：激活码永久绑定您的账号，支持在不同电脑上使用（需输入相同的账号和密钥）。

4. 快速开始

  1. 选择输入：点击 “Select Input Folder” 选择包含书法图片的文件夹。
  2. 设置输出：默认输出在软件目录下的 output 文件夹，您也可以点击 “Select Output Folder” 修改。
  3. 预览：点击左侧文件列表中的任意文件名，右侧会实时显示提取效果预览。
  4. 调整参数 (仅限专业版)：
    • 如果字断开了，增加 **Dilation (膨胀)**。
    • 如果字连在一起了，减小 **Merge Dist (合并距离)**。
    • 如果噪点太多，增加 **Min Area (最小面积)**。
  5. 开始处理：点击 Start Processing 按钮，软件将自动处理文件夹内的所有图片。

5. 常见问题

Q: 处理出来的字是白色的？
A: 软件默认处理“白纸黑字”。如果是黑底白字（碑帖），软件会自动尝试反转。如果效果不好，请确保图片对比度足够高。

Q: 大图片处理很慢？
A: 高分辨率图片（如 10000px 以上）处理时间较长是正常的。软件会自动优化参数以防止内存溢出。

Q: 遇到 Bug 怎么办？
A: 请截图报错信息或异常效果，联系开发者获取技术支持。
